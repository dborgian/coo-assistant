# Fix an Issue

Diagnose and fix a bug or issue in the COO Assistant.

Steps:
1. Understand the reported issue
2. Search relevant files using Grep/Glob
3. Read the affected service/module
4. Identify root cause
5. Apply fix with minimal changes
6. Run `npx tsc --noEmit` to verify
7. Test the fix if possible
8. Summarize what was changed and why
